## 第 1 步：推到 GitHub（约 1 分钟）

### 用 GitHub Desktop（推荐）

1. 打开 GitHub Desktop
2. **File → Add local repository** → 选择 `E:\xiangyin`
3. 应能看到 2 个提交待推送
4. 点顶部 **Push origin**
5. 登录 GitHub 账号（若提示）

### 或用 Cursor

1. 左侧 **源代码管理**
2. 点 **同步更改** / **Push**

完成后打开：https://github.com/zhuozhuodan75-ctrl/xingyin  
确认代码已更新。

### 设为公开（老师要求）

GitHub 网页 → 仓库 **Settings** → **Change visibility** → **Public**

## 第 2 步：Vercel 部署（约 3 分钟）

1. 打开 https://vercel.com ，用 GitHub 登录
2. **Add New → Project** → 选 `xingyin` → **Import**
3. 展开 **Environment Variables**，添加（值从本地 `.env` 复制）：

| 名称 | 说明 |
|------|------|
| `VITE_SUPABASE_URL` | Supabase 项目地址 |
| `VITE_SUPABASE_PUBLISHABLE_KEY` | Supabase 公钥 |
| `VITE_ADMIN_EMAIL` | 可选 |
| `VITE_ADMIN_PASSWORD` | 可选 |

4. 点 **Deploy**，等完成
5. 复制地址，如 `https://xingyin-xxx.vercel.app`

### 若登录失败

Supabase → **Authentication** → **URL Configuration**  
把 Vercel 地址加到 **Redirect URLs**。
